GitHub-hosted CMS Admin

Панель управления через GitHub Actions

Эта версия админки работает как статическая GitHub Pages-страница. Изменения контента отправляются в защищенный workflow, коммитятся в репозиторий и затем деплоятся статическим артефактом на VPS.

GitHub доступ

Введите fine-grained token с правами Actions: write и Contents: write для репозитория. Токен хранится только в sessionStorage текущего браузера.
